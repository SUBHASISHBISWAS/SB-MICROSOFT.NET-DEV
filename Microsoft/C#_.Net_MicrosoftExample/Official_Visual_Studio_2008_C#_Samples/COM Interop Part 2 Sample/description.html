索引子範例

這個範例介紹 C# 類別如何能宣告索引子以提供陣列形式來存取類別。

安全性注意事項

這個程式碼範例僅供概念說明之用，同時因為範例中的程式碼撰寫並沒有周全地顧及所有安全性問題，所以本範例不適用於實際執行的應用程式或網站中。如果使用者擅自將範例程式碼用於其他意圖而導致連帶事件或重大損害時，Microsoft 將不承擔任何責任。

若要建置和執行 Visual Studio 的索引子範例

  1. 在 [方案總管] 中，以滑鼠右鍵按一下 [indexers] 專案，然後按一下 [屬性]。

  2. 開啟 [組態屬性] 資料夾，並按一下 [偵錯]。

  3. 在 [命令列的引數] 屬性中，輸入 ..\..\Test.txt。

  4. 按一下 [確定]。

  5. 從 [偵錯] 功能表中按一下 [啟動但不偵錯]。

若要從命令列建置和執行索引子範例

  1. 若要編譯範例程式，請在命令提示字元中輸入如下命令：

    csc indexer.cs

  2. 這個範例程式將命令列引數所指定檔案中的位元組反向。例如，若要將 Test.txt 中的位元組反向並查看結果，請發出以下命令：

    indexers Test.txt
type Test.txt

  3. 若要將反向的檔案還原成正常，請針對同一個檔案再次執行程式。
