Esempio di restituzione

In questo esempio viene illustrato come creare una classe di tipo elenco che implementa IEnumerable<int> e la parola chiave yield per attivare l'iterazione foreach sul contenuto dell'elenco. Vengono definite due proprietà, una delle quali restituisce i numeri dispari, mentre l'altra restituisce i numeri pari.

Nota sulla sicurezza

Il codice di esempio viene fornito per illustrare un concetto e non deve essere utilizzato in applicazioni o siti Web in quanto potrebbe non rappresentare le procedure di codifica più sicure. Microsoft esclude qualsiasi responsabilità di danni diretti o indiretti qualora il codice di esempio venga utilizzato per scopi diversi da quelli a cui è destinato.

Per compilare ed eseguire l'esempio di codice di restituzione in Visual Studio

  • Scegliere Avvia senza eseguire debug dal menu Debug.

Per compilare ed eseguire l'esempio di codice di restituzione dalla riga di comando

  1. Utilizzare il comando Cambia directory (cd) per passare alla directory Yield.

  2. Digitare quanto segue: 

    csc Yield.cs
